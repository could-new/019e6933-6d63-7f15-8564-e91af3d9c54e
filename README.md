# تطبيق تعلم الكورية للأطفال

تطبيق ممتع وتفاعلي مصمم خصيصًا لمساعدة الأطفال على تعلم أساسيات اللغة الكورية بطريقة مبسطة وجذابة.

## المميزات
- **واجهة مستخدم ملونة وجذابة:** تصميم مرح ومناسب للأطفال يسهل التفاعل معه.
- **فئات متعددة للتعلم:** يشمل التطبيق تعلم الأرقام، الألوان، الحيوانات، وأفراد العائلة.
- **كلمات كورية مع النطق وطريقة الكتابة:** عرض الكلمة الكورية مع طريقة نطقها بحروف عربية أو لاتينية، ومعناها باللغة العربية.
- **متوافق مع مختلف الشاشات:** يعمل بسلاسة على الهواتف والأجهزة اللوحية (الآيباد)، وحتى على أجهزة سطح المكتب.

## الأقسام المتوفرة
1. **الأرقام:** من 1 إلى 4.
2. **الألوان:** الألوان الأساسية (أحمر، أزرق، أصفر، أخضر).
3. **الحيوانات:** حيوانات أليفة ومألوفة.
4. **العائلة:** مسميات أفراد العائلة.

## التكنولوجيا المستخدمة
- **Flutter:** إطار عمل واجهة المستخدم.
- **Dart:** لغة البرمجة.

## كيفية التشغيل
للبدء في تشغيل المشروع، تأكد من تثبيت Flutter في بيئتك، ثم قم بتنفيذ الأوامر التالية في سطر الأوامر:
```bash
flutter pub get
flutter run
```
