# تطبيق تعلم الكورية للأطفال 🇰🇷✨

تطبيق تفاعلي وملون مصمم خصيصاً لمساعدة الأطفال على تعلم المفردات الأساسية باللغة الكورية بطريقة ممتعة وبسيطة.

## المميزات
- **أقسام تعليمية شاملة:** يحتوي التطبيق على 4 تصنيفات رئيسية (الأرقام، الحيوانات، الملابس، الألوان).
- **ثلاث لغات:** عرض الكلمة باللغة العربية، الإنجليزية، والكورية (مع نطق الكلمة باللغة العربية لتسهيل القراءة).
- **واجهة صديقة للطفل:** ألوان زاهية، أيقونات جذابة، وتصميم سلس يدعم تجربة التعلم.
- **تجاوب كامل:** يدعم شاشات الهواتف والأجهزة اللوحية وتجربة الويب.

## المحتوى التعليمي
1. **الأرقام (Numbers):** من 1 إلى 10 وما فوق، مع النطق الكوري والأجنبي.
2. **الحيوانات (Animals):** مجموعة واسعة من الحيوانات مثل القطط، الكلاب، الفيلة، الأسود، والمزيد.
3. **الملابس (Clothes):** قطع الملابس الأساسية كالقمصان، السراويل، الفساتين، والأحذية.
4. **الألوان (Colors):** جميع الألوان الأساسية مع عرض لون البطاقة وفقاً للون المختار لربط الكلمة باللون.

## طريقة التشغيل

للعمل على هذا التطبيق محلياً:

1. تأكد من تثبيت [Flutter](https://flutter.dev/).
2. قم بعمل استنساخ (Clone) للمشروع.
3. قم بتشغيل الأمر التالي لجلب الحزم:
   ```bash
   flutter pub get
   ```
4. قم بتشغيل التطبيق:
   ```bash
   flutter run
   ```
